# Keys

This file provides a list of all available keys that can be used with the `MouseDown`, `MouseUp` and `TypeKeys` functions available in the [Keyboard API](API.md#api-keyboard).

While it is possible to emulate mouse clicks using this API, unless you have a specific reason for doing it this way, you should use the [Cursor API](API.md#api-cursor).
